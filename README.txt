--- What is it?
Scriptures is an android application for reading the KJV Bible or the LDS Standard Works. 

See http://blog.digitalbias.com/?p=139 for more details on what has happened in the last couple versions and why, as well as how to fix common problems.

--- Features

* Bookmarks
* Not locked into a single translation of the bible.
* Gesture navigation
* Adjustible font size
* Accelorometer when reading verses
* Navigation bread-crumb trail

--- Planned

* Search
* Notes and Highlighting
* More translations
* Tool for creating your own content
